# User Flows

## Flujo F1: Alta y acceso
SC-02 Registro → SC-03 OTP → SC-04 Login → SC-05 Dashboard.
- RF: RF-01, RF-02.
- BR: BR-08 (si OTP/proveedor requiere firma externa).

## Flujo F2: Reserva con copago
SC-06 Buscar → SC-07 Profesional → SC-08 Agenda → SC-09 Confirmar → SC-10 Pagar → SC-11 Reserva exitosa.
- RF: RF-04, RF-05, RF-06.
- BR: BR-01, BR-02, BR-03.

## Flujo F3: Gestión post-reserva
SC-12 Mis turnos → SC-13 Detalle → SC-14 Reprogramar / SC-15 Cancelar.
- RF: RF-07, RF-08.
- BR: BR-04, BR-05.

## Flujo F4: Resultados y documentos
SC-19 Resultados → SC-20 Detalle → SC-21 Documentos → SC-22 Compartir.
- RF: RF-09.
- BR: BR-06.

## Flujo F5: Grupo familiar
SC-27 Grupo familiar → SC-28 Alta integrante → SC-29 Cambiar titular activo.
- RF: RF-03.

## Flujo F6: Operación admin
Admin módulos de turnos/pagos/documentos/auditoría.
- RF: RF-11, RF-12.
- RNF: RNF-03, RNF-04, RNF-06.
